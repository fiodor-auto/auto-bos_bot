# 🚗 Auto Boss Bot

**Telegram бот для автомехаников** с функциями расшифровки VIN, поиска проблем и решений, а также поиска запчастей.

## ✨ Возможности

- 🔍 **VIN Расшифровка** - полная информация об автомобиле через NHTSA API
- 🔧 **База проблем** - поиск и добавление решений для распространённых проблем
- 🛠️ **Поиск запчастей** - интеграция с источниками запчастей
- 📰 **Новости авто** - последние новости об автомобилях через RSS
- 👥 **Управление пользователями** - отслеживание статистики
- 📊 **SQLite БД** - сохранение всех данных локально

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- pip

### Установка

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/fiodor-auto/auto-bos_bot.git
cd auto-bos_bot

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Создайте файл .env
cp .env.example .env

# 4. Отредактируйте .env и добавьте ваш TOKEN
nano .env
```

### Запуск

```bash
python main.py
```

## 📋 Команды

| Команда | Описание | Пример |
|---------|---------|--------|
| `/start` | Запуск бота и информация | `/start` |
| `/help` | Справка по всем командам | `/help` |
| `/vin <номер>` | Расшифровка VIN номера | `/vin WBA3A5C52EP600000` |
| `/problem <марка> <модель>` | Поиск проблем по марке/модели | `/problem BMW 3Series` |
| `/addproblem` | Добавить новую проблему и решение | `/addproblem` |
| `/search <запчасть>` | Поиск запчастей | `/search тормозные колодки` |
| `/rss` | Получить последние новости авто | `/rss` |

## 💾 База данных

Бот использует SQLite с двумя таблицами:

### problems
```sql
id          - ID записи (PRIMARY KEY)
make        - Марка автомобиля (TEXT)
model       - Модель автомобиля (TEXT)
year        - Год выпуска (TEXT)
problem     - Описание проблемы (TEXT)
solution    - Решение проблемы (TEXT)
added_by    - Имя автора (TEXT)
date        - Дата добавления (TEXT)
```

### users
```sql
id          - ID записи (PRIMARY KEY)
user_id     - ID пользователя Telegram (INTEGER UNIQUE)
username    - Имя пользователя (TEXT)
first_name  - Имя (TEXT)
joined_date - Дата присоединения (TEXT)
```

## 📚 Используемые API и библиотеки

- **aiogram** - асинхронная библиотека для Telegram Bot API
- **aiohttp** - асинхронные HTTP запросы
- **NHTSA VIN API** - расшифровка VIN номеров
- **feedparser** - парсинг RSS новостей
- **SQLite** - локальная база данных
